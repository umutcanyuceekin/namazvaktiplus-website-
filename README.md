# NamazVakti+ Web Sitesi

Bu klasör, NamazVakti+ uygulamasının tanıtım sayfasını içerir.

## İçerik
- `index.html`: Ana sayfa tasarımı.
- `style.css`: Stil dosyası.
- `app-ads.txt`: AdMob doğrulaması için gerekli dosya.

## Yayına Alma (GitHub Pages)

1. Yeni bir GitHub deposu oluşturun (Örn: `namazvaktiplus`).
2. Bu klasördeki dosyaları o depoya yükleyin.
3. GitHub depo ayarlarından **Settings > Pages** kısmına gidin.
4. "Deploy from a branch" seçeneğini ve `main` branch'ini seçip kaydedin.
5. Siteniz `https://[kullanıcı-adınız].github.io/namazvaktiplus/` adresinde yayına girecektir.
